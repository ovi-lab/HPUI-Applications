# HPUI-Applications {#hpui-applications}

This project [HPUI-Applications](https://github.com/ovi-lab/HPUI-Applications) showcases applications that are built using HPUI. This depends on [HPUI-Core](https://github.com/ovi-lab/HPUI-Core), [HPUI-Cone-Interactor](https://github.com/ovi-lab/HPUI-Cone-Interactor) and [HPUI-Components](https://github.com/ovi-lab/HPUI-Components). HPUI-Core, HPUI-Cone-Interactor and HPUI-Components need to be added as a dependency for a project before adding this.

For the documentation visit [HPUI-Docs](https://ovi-lab.github.io/HPUI-Docs)
